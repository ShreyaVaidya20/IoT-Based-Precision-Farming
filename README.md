# IoT-Based-Precision-Farming

Precision Farming using IoT is a smart agriculture solution designed to monitor soil and environmental conditions in real time using an ESP32 microcontroller. The system integrates a soil moisture sensor, DHT11 temperature and humidity sensor, and an RS485-based NPK sensor to continuously collect data related to soil health and environmental conditions. The collected information is transmitted over Wi-Fi to Firebase Realtime Database, where it can be accessed remotely through a web dashboard.

The project enables farmers to monitor important agricultural parameters such as temperature, humidity, soil moisture, and Nitrogen (N), Phosphorus (P), and Potassium (K) levels without being physically present in the field. By providing real-time insights into soil conditions, the system helps improve irrigation planning, nutrient management, and overall crop productivity while promoting efficient use of water and fertilizers.

The solution is built using ESP32, Arduino IDE, Firebase Realtime Database, and IoT technologies, making it a low-cost, scalable, and practical approach for modern precision agriculture. It demonstrates how embedded systems, cloud computing, and wireless communication can work together to support sustainable farming practices and data-driven agricultural decision-making.
